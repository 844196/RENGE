## Installation
### Mac OS X / Homebrew
```shellsession
$ brew tap 844196/Renge
$ brew install Renge
```

### Debian / apt-get
```shellsession
$ sudo apt-add-repository "deb http://ssiserver.moe.hm/debian ./"
$ sudo apt-get update
$ sudo apt-get install renge
```
